# Hermes 웹연구 품질게이트

자율 웹연구·지식 정제의 산출물을 *정량 점수*로 거르는 품질 게이트. mandate 갭 Priority 3(웹링크 지식 정제 자동화 — 헤르메스)의 설계 보강으로, [[에이전트_자는시간_학습루프]]의 헤르메스 *지식 스카우터*에 "얼마나 믿을 수 있는 소스인가"의 수치 컷오프를 더한다. 도구가 아니라 *채점 루브릭*만 차용한다.

## 채점 루브릭

| 축 | 기준 |
|---|---|
| 신뢰도 점수 | 0.0–1.0, **최소 0.6 / 목표 0.8** 미만 소스는 1차에서 탈락 |
| 멀티홉 깊이 | 엔티티 확장·인과 체인 추적 **≤5회** (무한 확산 차단) |
| 수집 전략 | planning-only / intent / unified 3종 분기 |
| 깊이 티어 | quick · standard · deep · exhaustive |

핵심은 *신뢰도 0.6 임계* — 볼트 입국심사가 추천도 ⭐로 정성 평가는 하되 수치화된 컷오프가 없던 지점을 메운다.

## 기존 게이트와의 위치

[[위키_저장_5대필터]]가 "올릴 가치가 있나"를 *정성*으로 묻는다면, 본 게이트는 funnel 더 앞단 — *수집 시점*에서 "이 소스를 들일 가치가 있나"를 *정량*으로 막는다. 저품질 원천을 [[맥락_오염_방지]] 이전에 조기 차단해, 정제 파이프라인([[move-to-raw]]→distill→compile)의 입력 순도를 높인다.

## 차용 경계

- 채점 *기준*만 차용 시 런타임 비용 0.
- 멀티홉 실행 도구(Tavily MCP 등) 도입은 별개 — 토큰·시간 비용이 붙으므로 본 노트는 *루브릭만* 채택한다.
- 깊이 티어는 distill 모드 분기(quick=요약 / exhaustive=전수)의 참조축으로도 쓸 수 있다.
