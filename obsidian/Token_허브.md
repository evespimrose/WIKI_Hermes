# Token 허브

토큰 통제와 워크플로우 영역의 모든 노트를 묶는 행성. 40개 위성의 진입점이며, 상위 항성 [[LLM_워크플로우_생태계]]에서 진입한다. 인접 행성은 [[RX_1_허브]]·[[Wiki_허브]].

## 행성 정의

LLM의 컨텍스트 윈도우를 유한 자원으로 다루기 위한 *모든 장치*가 여기 모인다. 문제 정의 → 설계 철학 → 실행 메커니즘 → 진화 이력의 4축이 한 행성 위에서 보인다.
