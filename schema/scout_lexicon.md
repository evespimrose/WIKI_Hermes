# scout_lexicon — 판례 단어장 (자가학습)

> 입국심사(`compile-wiki a`)의 *사람 판정*을 1줄/1~2단어로 누적하는 사례집. 각 판례가 키워드 가중치(+채용 / −반려)로 환산돼, 헤르메스 자율 스카우팅의 *방향타·사전필터*가 된다. `docs/Try_HermesKnowledgeScout` §2-1의 실체.
>
> **권한은 사람에게**: 본 단어장은 헤르메스의 사전필터일 뿐, 위키 승격의 최종 판정은 사용자 입국심사가 한다.

## 가중치 해석

- **+가중**: 해당 키워드가 제목/태그/본문에 짙으면 스카우트 후보로 *승격 우선*.
- **−가중**: 짙으면 *사전 탈락*(raw/_scout/에도 올리지 않음) → 비싼 입국심사 게이트 도달량 감축.
- **🟡혼합**: 추상 개념(+)은 추출하되 제품 설치·요금(−)은 버림 → distill 시 분리.

## 자가학습 규약

매 입국심사 후 판정을 본 표에 1행 append. 같은 −키워드가 누적되면 `vault_mandate.md` out-of-scope에 승격. 같은 +키워드가 군집을 이루면 신규 행성 후보로 [[LLM_Wiki_지식자산_현황]]에 보고.
